Upper-bound visual analysis: why clean 16x16 is hitting the wall

The current clean FutureSeed+RWKV+loop path is not primarily failing at rollout selection or Sudoku global consistency. It is still below the local-token reliability regime where exact-board solving can emerge.

1. The exact-solve wall is exponential in holes

Even under an optimistic independent-cell approximation, exact probability is roughly blank_acc ^ holes. For 24 blanks, a 50% exact board needs about 97.2% blank accuracy. The latest clean 16x16 runs are around 33-34% at h24, so exact=0 is not mysterious; it is mathematically forced.

Decision

The immediate ceiling is local reliability under clean scaling: 16x16 h24/h32 predictions are too wrong before Sudoku-level consistency can matter. FutureSeed decay and naive all-loop supervision are not enough. The next Bitter-Lesson-aligned work should improve generic modeling capacity/credit assignment or FutureSeed's learned recurrent state mechanism only if it can raise blank accuracy first.
