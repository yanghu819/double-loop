What fails without FutureSeed

On this case, no-FutureSeed is not just making isolated digit mistakes. Its final board violates global Sudoku coupling through duplicated digits in rows, columns, and boxes.

  • R1C3: noFS loop5=5, truth=3
  • R1C4: noFS loop5=5, truth=1
  • R4C4: noFS loop5=9, truth=7
  • R4C7: noFS loop5=4, truth=3
  • R5C2: noFS loop5=5, truth=8
  • R8C4: noFS loop5=1, truth=9

Conflict units in no-FutureSeed loop5:

  • row 1: 5@R1C3,R1C4,R1C7
  • row 4: 4@R4C7,R4C8; 9@R4C4,R4C6
  • row 5: 5@R5C2,R5C9
  • row 8: 1@R8C4,R8C6
  • col 2: 5@R5C2,R6C2
  • col 3: 5@R1C3,R9C3
  • col 4: 5@R1C4,R3C4
  • col 7: 4@R4C7,R8C7
  • box 1,1: 5@R1C3,R2C1
  • box 1,2: 5@R1C4,R3C4
  • box 2,1: 5@R5C2,R6C2
  • box 2,2: 9@R4C4,R4C6
  • box 2,3: 4@R4C7,R4C8
  • box 3,2: 1@R8C4,R8C6
